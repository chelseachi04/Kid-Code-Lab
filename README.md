# 🚀 Kid Code Lab - Interactive HTML Learning Platform

**Kid Code Lab** is a vibrant, gamified coding platform designed to teach children and teenagers the fundamentals of web development. By blending a friendly user interface with a real-time code editor, it turns learning HTML into an engaging adventure.



## ✨ Features

* **Code Quest (Learning Path):** A guided curriculum starting from **Module 1** that teaches the "skeleton" of a webpage step-by-step.
* **The Sandbox (Code Editor):** A free-play zone where kids can use all unlocked tags and characters to build whatever they imagine.
* **Real IDE Experience:** A custom code editor with auto-indentation, line breaks, and smart cursor placement to teach professional coding habits early.
* **Live Preview:** See your creations come to life instantly with a side-by-side live rendering engine.
* **Kid-Friendly UI:** Includes a "Fancy" modal system, colorful cartoon characters, and a guided tour to ensure no child gets lost.
* **Dark Mode:** A togglable theme to keep the coding lab comfortable at any time of day.

## 🛠️ Built With

* **React 19** - Frontend framework
* **Vite** - Build tool
* **Tailwind CSS** - Styling and responsive layout
* **Vercel** - Hosting and Deployment
* **Lucide React** - Iconography

* `src/components`: Reusable UI components like the Editor, Preview, and Modals.
* `src/data`: Game logic and module curriculum data.
* `src/assets`: Cartoon characters and mascots.
